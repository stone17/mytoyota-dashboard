Credentials Management

Securely store your MyToyota username and password. These are encrypted and stored locally.

MQTT Settings

Send vehicle data to an MQTT broker for home automation systems like Domoticz or Home Assistant.

Default is mytoyota/{vin}. Sensor data will be published to sub-topics like .../odometer.

Prefix for Home Assistant or Domoticz MQTT Auto Discovery. Leave as `homeassistant` unless you have a specific reason to change it.

Logging Settings

Configure logging behavior for the application.

Number of recent log lines to keep in memory for the Logs page. A restart is required for this change to take effect.

Location & GPS Settings

Configure how trip addresses and GPS data are resolved.

When enabled, the application will use an external service (Nominatim) to convert trip coordinates into street addresses.

Warning: Enabling this will significantly increase the time it takes to fetch trips and will use more database storage.

Import Trip History

Upload a trip history CSV file exported from the Toyota app. The filename must start with the vehicle's VIN (e.g., SB1..._2024-01-01_2024-12-31.csv).

Data Utilities

Perform one-time maintenance tasks on your data.

If you've just updated, click this to calculate imperial units (miles, MPG) for all your existing trips. This is a one-time operation.
